# Daxter_2.0

## An Accountant Data Aggregation and Insights Dashboard

### Overview
A unified dashboard for accountants by consolidating real-time financial and tax compliance data from multiple agent based sources. The platform integrates intelligent AI workflows, agent orchestration, and real-time data processing, requiring robust frontend, backend, and database. A core workflow simulating data ingestion from multipe agents, providing summary and query capabilities, and presenting it all in a user-friendly dashboard with basic AI interaction and observability.
